# wangEditor React TypeScript Demo

当前代码通过 [create-react-app](https://create-react-app.dev/) 创建，使用 TypeScript 和 React 18

## 技术栈

- React 18
- TypeScript 5.2
- Antd 5.9
- wangEditor 5.6
- Create React App

## 本地运行

```shell
pnpm install
pnpm dev
```

## 可用命令

- `pnpm dev` - 启动开发服务器（推荐）
- `pnpm start` - 启动开发服务器
- `pnpm build` - 构建生产版本
- `pnpm test` - 运行测试
- `pnpm eject` - 暴露webpack配置（不推荐）
